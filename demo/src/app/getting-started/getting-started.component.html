Getting Started

Easily configure a datacontext for all the applicaion entities with only one service.

Installation

Install it with npm

$ npm install --save ngx-datacontext

When using firebase database or firestore, install firebase dependencies 

$ npm install --save firebase angularfire2

Usage

Add NgxDataModule in the root module

NOTE: Firebase and Firestore should be registered as normal. Firebase Installation Guide

Entities

To create an entity do the following things:

Entity Decorator Use the entity decorator to identify the entity and that will be used in the endpoints or list/collection from firebase

@Entity('users')

Extend BaseEntity or FirebaseEntity

export class User extends FirebaseEntity

Attribute Decorator Use the attribute decorator to identify the attributes to serialize

    @Attribute() firstName: string;
